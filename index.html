소스 코드 링크

메인 페이지

저희 메인 페이지로 자신이 구독한 사이트의 최신 글, 옆의 날짜 설정을 통해 과거의 글도 불러올 수 있습니다.

  • 개발 박석주

구독 페이지

구독 페이지에서 RSS 기능을 하는 사이트들의 URL을 가져와 입력하여 구독이 가능합니다.

  • 개발자: 배상인

구독 사이트 정리 페이지

구독한 사이트들을 사이트 별로 모아서 볼 수 있습니다.

  • 개발자: 이강재

스크랩 페이지

스크랩을 통해 저장한 글들을 보여주는 곳 입니다.

  • 개발자: 배상인, 이강재
